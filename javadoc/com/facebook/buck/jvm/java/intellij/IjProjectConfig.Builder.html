    @NotThreadSafe
public static final class IjProjectConfig.Builder
extends Object
    Builds instances of type IjProjectConfig. Initialize attributes and then invoke the build() method to create an immutable instance.

    Builder is not thread-safe and generally should not be stored in a field or collection, but instead used immediately to create instances.
